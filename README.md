# MustacheNMansion

##### เกมของเราเป็นเกมเเนวปริศนา และสืบสวน ที่ผู้เล่นต้องไปสืบสวนคดีฆาตกรรมในคฤหาสน์แห่งหนึ่ง โดยสืบสวนจากเบาะเเสตามสถานที่ต่าง ๆ ในคฤหาสน์รวมถึงผู้ต้องสงสัย 

##### เกมนี้เป็นเกมทำระหว่างเรียนอยู่ชั้นปีที่ 2 มหาวิทยาลัยเชียงใหม่ อาจมีบัคและข้อผิดพลาดเล็กน้อย
