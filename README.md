# MustacheNMansion

##### เป็นเกม  2D แนวปริศนา และสืบสวนสอบสวน ที่ผู้เล่นต้องไปสืบสวนคดีฆาตกรรมในคฤหาสน์แห่งหนึ่ง โดยสืบสวนจากเบาะแสตามสถานที่ต่าง ๆ ในคฤหาสน์รวมถึงผู้ต้องสงสัย 

##### เกมนี้เป็นเกมที่ทำระหว่างเรียนอยู่ชั้นปีที่ 2 มหาวิทยาลัยเชียงใหม่ อาจมีบัคและข้อผิดพลาดที่เกิดขึ้นภายในเกม 

##### ใช้ Visual Studio(Monogame) ในการพัฒนาเกมนี้ขึ้นมา 

##### ใช้ .Net Core ในการรันตัวเกม
